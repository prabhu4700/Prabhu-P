This project is a simple Python program that demonstrates the concept of reciprocity.

Reciprocity means giving help to someone and receiving help in return. This idea is important in real life and in teamwork.

In this program, users help each other, and the program tracks how much help is given and received. It uses Python classes and objects to represent users and their actions. This project is ideal for beginners who want to learn Python basics and object-oriented programming (OOP).

Demonstrate real-life ideas using code

⚙️ Features
Create multiple users

🧩 How the Program Works
A User class is created with:
Name
Help given counter
Help received counter
The give_help() function:
One user helps another user
Updates both users’ help counts
The program simulates reciprocal actions:
Example: User1 helps User2, and User2 helps back
The show_status() function displays:
Total help given
Total help received

💻 Example Code
Python
user1 = User("Alice")
user2 = User("Bob")

user1.give_help(user2)
user2.give_help(user1)

user1.show_status()
user2.show_status()
